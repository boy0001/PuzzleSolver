How to use:
 - Open up cmd, and go to this directory
start.bat <file> <method>

Or to compile yourself:
 - Open up cmd, and go to this directory
javac -g com/boydti/puzzle/Main.java
java com.boydti.puzzle.Main <file> <method>

Methods:
BFS (Best First Search)
DFS (Depth First Search)
GBFS (Greedy Best First Search)
AS (A* Search)
CUS1 (Custom 1 - Random choice)
CUS2 (Custom 2 - bi-directional Manhattan)

To run the test on the 1000 randomly generated files:
 - Open up cmd and go to this directory
test.bat <method>
